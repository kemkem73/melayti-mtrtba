ملايتي مترتبة - صفحة الهبوط الاحترافية
================================================================================

تم إنشاء صفحة هبوط احترافية وجاهزة للنشر الفوري لمنتج طقم تثبيت الملاية.

================================================================================
محتويات المشروع
================================================================================

index.html          - ملف HTML الرئيسي للصفحة
styles.css          - ملف CSS للتنسيق والتصميم
script.js           - ملف JavaScript للتفاعلية
assets/             - مجلد الصور والملفات الإضافية
README.txt          - هذا الملف

================================================================================
المميزات المضمنة
================================================================================

✓ تصميم احترافي وجذاب بالكامل
✓ متوافق مع جميع الأجهزة (Responsive Design)
✓ دعم اللغة العربية (RTL)
✓ تحسين محركات البحث الأساسي (SEO)

================================================================================
كيفية النشر
================================================================================

الخيار 1: Netlify (الأسهل والأسرع)
─────────────────────────────────────
1. اذهب إلى https://netlify.com
2. سجل دخول أو أنشئ حساب مجاني
3. اسحب وأفلت مجلد المشروع على صفحة Netlify
4. سيتم نشر الموقع فوراً وستحصل على رابط عام

الخيار 2: Vercel
─────────────────
1. اذهب إلى https://vercel.com
2. سجل دخول بحسابك
3. اختر "Import Project"
4. اختر المجلد المحلي أو GitHub
5. انقر "Deploy"

الخيار 3: GitHub Pages
──────────────────────
1. أنشئ مستودع GitHub جديد
2. ارفع ملفات المشروع
3. اذهب إلى Settings > Pages
4. اختر Branch: main
5. سيتم النشر تلقائياً

الخيار 4: استضافة تقليدية (FTP)
────────────────────────────────
1. احمل جميع الملفات عبر FTP
2. تأكد من أن index.html في المجلد الجذر
3. تأكد من أن مجلد assets موجود مع الصور

الخيار 5: اختبار محلي
─────────────────────
1. انقر مزدوج على index.html
2. ستفتح الصفحة في المتصفح مباشرة
3. جميع الميزات تعمل بدون خادم

الملفات المهمة

================================================================================
تخصيص الصفحة
================================================================================

تغيير الألوان:
──────────────
افتح styles.css وابحث عن :root
غير القيم التالية:
- --primary-color: #e74c3c (اللون الأحمر الأساسي)
- --secondary-color: #3498db (اللون الأزرق الثانوي)

تغيير رقم الواتساب:
───────────────────
ابحث عن "01207327331" في index.html
استبدله برقمك الخاص
تأكد من إضافة رمز الدولة (+20 للمصر)

تغيير الأسعار:
──────────────
ابحث عن "325 ج.م" في index.html
استبدل بالسعر الجديد
افعل الشيء نفسه للعروض الأخرى

تغيير الصور:
────────────
استبدل الصور في مجلد assets بصورك الخاصة
تأكد من الاحتفاظ بنفس أسماء الملفات
أو حدث المسارات في index.html
